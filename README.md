# Библиотека коррупционных кейсов

Небольшой сайт для просмотра, поиска и систематизации коррупционных кейсов.

## Что есть в проекте

На сайте есть:

- публичная часть с кейсами;
- поиск и фильтры;
- карточка отдельного кейса;
- административная панель;
- импорт кейсов из `.docx`.

## Что нужно для запуска

Нужен установленный **Python 3**.

## Запуск на Linux

Открой терминал в папке проекта и выполни:

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
python app.py
```

После запуска сайт будет доступен по адресу:

```text
http://127.0.0.1:8080
```

## Запуск на Windows

### Вариант 1. PowerShell

Открой PowerShell в папке проекта и выполни:

```powershell
py -m venv .venv
.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
python app.py
```

### Вариант 2. Command Prompt (cmd)

Открой командную строку в папке проекта и выполни:

```bat
py -m venv .venv
.venv\Scripts\activate.bat
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
python app.py
```

После запуска сайт будет доступен по адресу:

```text
http://127.0.0.1:8080
```

## Как открыть сайт

Открой в браузере:

```text
http://127.0.0.1:8080
```

## Как войти в админку

Страница входа администратора:

```text
http://127.0.0.1:8080/admin/login
```

Логин и пароль по умолчанию:

```text
admin
admin123
```

## Что можно делать в админке

В административной панели можно:

- создавать новые кейсы;
- редактировать существующие;
- менять статус кейса;
- импортировать материалы из Word;
- редактировать страницу «О проекте»;
- управлять справочниками.

## Основные разделы сайта

На сайте есть три раздела:

- Россия
- Иностранные государства
- Международные организации

## Важные файлы проекта

- `app.py` — основной файл приложения;
- `requirements.txt` — зависимости Python;
- `seed_cases.json` — стартовые данные кейсов;
- `static/style.css` — стили сайта.
